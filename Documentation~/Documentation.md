# Poi to Quest Hue Converter

**Version:** 1.1.6  
**Author:** Terrorized  
**Unity Tool for VRChat Avatar Optimization**

---

## Overview

Poi to Quest Hue Converter is a Unity Editor tool to convert Poiyomi shader hue shift animations to Quest compatible VRChat hue shift properties.

- Converts Poiyomi hue shift properties to VRChat Quest-compatible properties
- Automatically scales values from Poiyomi range (0-1) to VRChat range (0-6.28)
- Scans and modifies FX layer animations
- Optionally enables shader keywords on materials used in animations
- Provides a visual list of all animations that will be affected

## Instructions

### License Activation

1. Open the tool from `Terrorized → Tools → PoiToQuestHue`
2. Enter your Payhip license key in the "License Key" field
3. Click "Verify License"
4. Once verified, the tool interface will unlock

### Basic Usage

1. **Select Avatar**: Drag your VRC Avatar Descriptor into the "VRC Avatar" field
2. **Review Animations**: The tool will automatically scan and display all animations with hue shift properties
3. **Configure Options** (Optional):
   - Enable "Enable VRC Hue Shift on Animated Materials" if you want shader keywords enabled automatically
4. **Convert**: Click the "Convert Hue Shift Properties" button
5. **Done**: All animations will be updated with the new properties and scaled values
